# Assets

Place the following files here before building:

- `icon.png` — 1024×1024 app icon
- `splash.png` — 1284×2778 splash screen image
- `adaptive-icon.png` — 1024×1024 Android adaptive icon foreground
- `favicon.png` — 48×48 web favicon
- `notification-icon.png` — 96×96 Android notification icon (white on transparent)

## Fonts

Place in `assets/fonts/`:

- `SpaceGrotesk-Regular.ttf`
- `SpaceGrotesk-Medium.ttf`
- `SpaceGrotesk-Bold.ttf`
- `JetBrainsMono-Regular.ttf`

Download Space Grotesk from: https://fonts.google.com/specimen/Space+Grotesk  
Download JetBrains Mono from: https://www.jetbrains.com/lp/mono/
